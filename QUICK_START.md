## 🚀 Getting Started (5 Minutes)

### Step 1: Open Website Locally
```
Option A: Double-click any .html file
Option B: Use Python: python -m http.server 8000
Option C: Use Node: npx http-server
```

### Step 2: Test All Pages
- Click through all pages and links
- Check mobile view (press F12, then toggle device toolbar)
- Verify all content displays correctly

### Step 3: Customize Content (Optional)
- Update company name (search "Velonixs")
- Update email (search "info@velonixs.com")
- Update domain (search "velonixs.com")
- See CUSTOMIZATION.md for details

### Step 4: Deploy Online
- Choose hosting: Vercel, Netlify, or traditional hosting
- See DEPLOYMENT.md for step-by-step instructions
- Takes 5-10 minutes

## 📝 Quick Customization (What to Change)

### Most Important Updates

**1. Company Email**
- Search: `info@velonixs.com`
- Replace with: your actual email
- In all files: index.html, services.html, about.html, contact.html

**2. Company Name**
- Search: `Velonixs` 
- Replace with: your company name
- In all files

**3. Domain Name**
- Search: `velonixs.com`
- Replace with: your domain
- In footer and links

**4. Logo/Images**
- Add your logo to: `assets/images/`
- Update navbar to use logo image

**5. Color Theme** (Optional)
- File: `assets/css/style.css`
- Find: `--primary-color: #0056b3;`
- Change to your brand color

### Services Content
- Edit service descriptions in `services.html`
- Update capabilities and benefits
- Add/remove services as needed

### About Us
- Update company description in `about.html`
- Modify vision, mission, values
- Update technology stack

## 📧 Contact Form Setup

The contact form currently shows a success message locally. 

For production, choose ONE:

### Option 1: Formspree (Easiest)
1. Go to formspree.io
2. Create form and get ID
3. Update form in contact.html (1 line change)
✅ Emails go to your inbox!

### Option 2: Netlify Forms
1. Deploy to Netlify (see DEPLOYMENT.md)
2. Add netlify attribute to form
✅ Automatic form handling!

### Option 3: EmailJS (JavaScript)
1. Sign up at emailjs.com
2. Add JavaScript code to main.js
✅ Emails via JavaScript!
